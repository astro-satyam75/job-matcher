# 🌍 Remote Job Matcher (ML Project)

## Overview

This project is a machine learning-based system that matches resumes with remote job descriptions and identifies skill gaps.

## Features

* Job matching using TF-IDF and cosine similarity
* Skill gap detection
* Market insights dashboard
* Interactive Streamlit UI

## Tech Stack

* Python
* SQL (SQLite)
* scikit-learn
* Streamlit

## How it works

1. Resume and job descriptions are converted into vectors using TF-IDF
2. Cosine similarity is used to compute match scores
3. Keyword-based scoring improves ranking
4. Skill gaps are identified by comparing extracted skills
